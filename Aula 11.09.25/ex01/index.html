Condicionais em Programação

O que é uma condição?

Uma condição é uma comparação que pode resultar em verdadeiro (true) ou falso (false).

Exemplo de condições:

idade >= 18 → Verdadeiro se a idade for 18 ou mais

temperatura < 0 → Verdadeiro se a temperatura for negativa

senha == "1234" → Verdadeiro se a senha for "1234"

Bloco de Código

É um conjunto de instruções dentro de { } que só será executado se a condição for atendida.

Estrutura básica do if (se):

if (condição) {
    // Este código só executa
    // se a condição for VERDADEIRA
}

Estrutura if-else (se-senão):

if (condição) {
    // Executa se a condição for VERDADEIRA
} else {
    // Executa se a condição for FALSA
}

Condicionais Múltiplas

Podemos testar várias condições usando else if:

if (idade < 12) {
    console.log("Criança");
} else if (idade < 18) {
    console.log("Adolescente");
} else if (idade < 60) {
    console.log("Adulto");
} else {
    console.log("Idoso");
}
