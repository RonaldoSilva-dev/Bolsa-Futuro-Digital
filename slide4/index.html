Nesta aula 4 de HTML5 conhecemos inúmeras informações, as quais apreseta-se aqui na forma que se segue:

Conceito de HTML: de forma sucinta pode-se conceber como sendo uma linguagem de marcação de texto. Noutras palavras, é a linguagem de que se faz uso a fim de estruturar páginas que serão visualiazadas na web.

Para tanto, conhecemos a estrutura elementar que uma página HTML tem que ter. E é sobre ela que irá se falar. Vejam a imagem abaixo:

Ao lado de cada elemento ou de cada linha, há numeração das linhas, a qual é importante para que se possa fazer referência a cada elemento.

Disto isto, iremos nos referir a cada elemento com base na numeração das linhas.

Na linha 1, temos a declaração do tipo de documento, que é o HTML5. Esta declaração é obrigatória e deve ser a primeira linha do código. Noutras palavras, é a linha que diz ao navegador que o documento é um arquivo HTML5, e que o interprete de acordo com suas diretrizes.

Na linha 2, temos a abertura do elemento html, o qual envolve todo o conteúdo da página. Este elemento também é obrigatório e deve ser o segundo elemento do código. Note que este elemento possui um atributo chamado lang, o qual indica o idioma da página. No caso, pt-br indica que a página está em português do Brasil. Noutras palavras, este atributo é importante para que os mecanismos de busca e os leitores de tela possam identificar o idioma da página.

Na linha 3, temos a abertura do elemento head, o qual contém informações sobre o documento, como o título, os metadados etc. Este elemento também é obrigatório e deve ser o terceiro elemento do código com sua respectiva tag de fechamento na linha 7.

Na linha 4, temos a tag meta com atributos charset que também é seguida de seu valor, que é UTF-8. Esta tag é importante para definir a codificação de caracteres do documento, garantindo que os caracteres especiais sejam exibidos corretamente. Noutras palavras, esta tag é essencial para que o navegador interprete corretamente os caracteres do texto, exemplo, os acentos gráficos, crase etc.

Na linha 5, temos a tag meta com atributos name e content e seus respectivos valores informa ao navegador como deve ser feita a exibição da página em dispositivos móveis. O valor width=device-width indica que a largura da página deve ser igual à largura da tela do dispositivo, e o valor initial-scale=1.0 indica que o zoom inicial deve ser 1. Noutras palavras, esta tag é fundamental para garantir que a página seja responsiva e se adapte a diferentes tamanhos de tela.

Na linha 6, temos a tag title, a qual define o título da página que será exibido na aba do navegador. Noutras palavras, é aqui que se informa o nome do site ou da página. É aquela frase que aparece na aba do navegador quando se abre uma página.

Na linha 7, trata-se da tag de fechamento do elemento head, o qual encerra as informações sobre o documento que se iniciou na linha 3.

Na linha 8, temos a abertura do elemento body, o qual contém o conteúdo visível da página (usuário vê), como textos, imagens, links etc. Este elemento também é obrigatório e deve ser o quarto elemento do código com sua respectiva tag de fechamento na linha 17.

Na linha 9, temos a tag h1, a qual define um título de nível 1. Este é o título mais importante da página e deve ser usado para o título principal. Noutras palavras, é o título que resume o conteúdo da página.

Na linha 10, temos a tag h2, a qual define um título de nível 2. Este é o segundo título mais importante da página e deve ser usado para subtítulos. Noutras palavras, é o título que complementa o título principal.

Na linha 11, temos a tag p, a qual define um parágrafo de texto. Este é o elemento usado para agrupar blocos de texto. Noutras palavras, é o elemento que contém o texto que será exibido na página. Noutras palavras, é o elemento que contém o texto que será exibido na página, conforme estamos a fazer aqui para cada linha.

Na linha 12, temos a tag img, a qual define uma imagem. Esta tag possui os atributos src e alt. O atributo src indica o caminho da imagem, e o atributo alt fornece um texto alternativo que será exibido caso a imagem não possa ser carregada. Noutras palavras, esta tag é usada para inserir imagens na página.

Na linha 13, temos a tag de fechamento do elemento body, o qual encerra o conteúdo visível da página que se iniciou na linha 8.

Na linha 14, temos a tag de fechamento do elemento html, o qual encerra todo o conteúdo da página que se iniciou na linha 2.

Bem! Inúmeras outras tags foram vistas nessas 4 aulas, exemplo, as tittle, target, rel, href etc. Contudo, se fosse citar aqui e explicar a página ficará extremamente grande. Preferiu-se abordar tags estruturais do HTML, bem como outras que apareceram no momento do print que se fez.
